# lofar-RFI-detection
This repository serves the purpose of satellite RFI detection in LOFAR station in Irbene, Latvia. It includes:

- lofar-rfi: developed software for Fraunhofer distance plot and sensitivity calculations (including uncertainty error).
- lofarimaging: updated LOFAR pipeline with filtering methods for satellite detection.
- nf-testing: part of 'lofarimaging' code regarding near-field imaging, for faster evaluation when near-field is the sole focus of the experiment.
- lofarimaging-ilisa: similar code to 'lofarimaging' but using iLiSa as the main calibration software.
